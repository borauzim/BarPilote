# Assets 3D - Boissons BarPilote 🍏🥤✨

Ce dossier est destiné à recueillir les visuels 3D, les rendus photoréalistes et les photos détourées des boissons qui seront affichées sur le site et l'application BarPilote.

### 💡 Recommandations de style :
- **Format** : PNG (détouré/transparent) ou WebP pour le Web.
- **Esthétique** : Style Apple (Lumière douce, reflets léchés, condensation réaliste).
- **Couleurs** : Prioriser les rendus qui s'accordent avec la palette **Orange Vibrant** de BarPilote.

*Déposez vos fichiers ici pour l'intégration future.*
