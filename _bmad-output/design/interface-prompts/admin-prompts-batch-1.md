# BarPilote : Design Prompts - Batch Admin (Responsable) 🍏🏗️

**Style Global** : Minimalisme "Apple-style". Palette de couleurs : Blanc pur (SF Pro White), Gris sidéral (Space Gray), Bleu San Francisco. Typographie : SF Pro Display. Espaces blancs généreux, bords arrondis (24px), ombres portées douces, flou de verre (Glassmorphism) pour les overlays.
