# BarPilote : Design Prompts - Batch Reporting & Data 🍏📊📉

**Style Global** : Apple "Analytical" Design. Information dense mais respiration maximale. Données présentées sous forme de "Cartes de Performance". Utilisation de dégradés subtils pour les graphiques, typographie SF Pro Mono pour les chiffres financiers. Clarté chirurgicale.

---

## 📈 01. Analyses de Ventes (Sales Analytics)

### Prompt 121 : Performance Hebdomadaire (Bar Chart)
"Design a minimalist weekly performance screen. A series of 7 vertical bars (Mon-Sun). High bars are vibrant blue, low bars are light gray. Summary text: 'Revenue this week: +15% vs last week'. Clean white background, no distracting grid lines."

### Prompt 122 : Répartition des Ventes par Catégorie (Donut Chart)
"A sleek donut chart view. Blue for Beers (60%), Gold for Spirits (25%), Green for Softs (15%). Small minimalist legend below. Center of donut shows: 'Total: $1,450'. Apple Fitness app style aesthetics."

### Prompt 123 : Top des Ventes - Le Podium des Produits
"List view of top 5 bestsellers. High-quality product icons on the left. Name and count (e.g., 'Heineken: 450 units'). A small sparkline (mini graph) next to each item showing trend. Typography-focused."

### Prompt 124 : Heatmap des Heures de Pointe (Peak Hours)
"A grid representation of the day (24 hours). Cells are shaded from light blue to deep navy based on sales volume. Peak hour highlighted: '23:00 - High Activity'. Extremely clean, professional dashboard view."

## 👥 02. Performance du Personnel (Staff KPI)

### Prompt 125 : Dashboard de Performance Serveur (Detailed)
"Detail view for a specific server (e.g., 'Tshala'). 3 large cards at top: 'Service Speed (4m)', 'Total Sales ($450)', 'Table Turn-rate (45m)'. Below, a list of last 20 orders served. Apple-style cards with soft shadows."

### Prompt 126 : Comparaison de Performance Staff
"Side-by-side comparison screen for 2 servers. Parallel metrics shown with simple bar indicators. Goal: Identify the 'Master of the Crate' of the week. Minimalist, motivating UI. Soft gray background."

### Prompt 127 : Taux de Pourboires (Insights)
"Privacy-focused screen showing average tips per table for a server (if reported). 'AVG Tip: 1,500 FC'. Graph showing tip trends over the shift. Clean, supportive UI. SF Pro typography."

### Prompt 128 : Journal des Erreurs/Annulations par Serveur
"Admin audit view. List of cancelled orders linked to a server. 'Table 4: Cancelled by Admin - Reason: Wrong item'. Transparent data display. No red alerts, just factual gray list."
