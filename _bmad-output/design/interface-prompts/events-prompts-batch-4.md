# BarPilote : Design Prompts - Batch Évènementiel (Events) 🍏💒🏢

**Style Global** : Apple "Events" Aesthetics. Élégant, solennel mais fonctionnel. Utilisation de palettes de couleurs adaptées (Or doux, Blanc cassé, Noir profond). Focus sur la logistique de masse et la précision chirurgicale du service de protocole.

---

## 🏗️ 01. Initialisation de l'Évènement (Organisateur)

### Prompt 061 : Dashboard de l'Évènement (Overview)
"Main control board for a wedding organizer in BarPilote. Top section: 'Event: Marriage Jean & Marie'. Stats cards: 'Stock Consumption: 35%', 'Active Protocols: 5', 'Guests Estimated: 300'. Aesthetic: Clean Apple layout, spacious, premium. Background is a soft white with subtle floral watermark."

### Prompt 062 : Création de la Carte des Boissons (Event Menu)
"Interface to select drinks for the event. List of products with checkboxes. Special input field: 'Total Quantity Allocated' for each brand (e.g., '100 Bottles of Veuve Clicquot'). Toggle: 'Unlimited Loop' for VIP tables. Design: List-based, high-quality icons."

### Prompt 063 : Plan de Salle Interactif (Table Setup)
"Visual grid for organizing tables in a wedding hall. Ability to name tables (e.g., 'Table Honeymoon', 'Table Family'). Each table shows a status icon: 'Needs refill' or 'Full'. Apple Maps style clean interface. Light gray cards on white."

## 👥 02. Gestion du Personnel & Affectations

### Prompt 064 : Liste du Personnel de Protocole
"Roster of protocol staff for the event. Cards with names, photos, and current status. Metric: 'Tables Assigned: 4'. Action: 'Message Staff'. Clean, typography-focused, Apple Health style detail view."

### Prompt 065 : Écran d'Affectation Dynamique (Organizer)
"Drag-and-drop interface to assign a server (e.g., 'Grace') to a group of tables. Visual link (blue line) connecting the staff avatar to the tables. Legend: 'Tshala: Tables 1-5', 'Grace: Tables 6-10'. Extremely intuitive, professional Apple tool design."

---

## 🍾 03. Suivi du Stock "Zéro Facture"

### Prompt 066 : État de la Cave Évènementielle (Live Stock)
"Real-time inventory for the organizer. 'Bottles Opened' vs 'Bottles Sealed'. Progress bar for each brand. Alert: 'Champagne running low in the Chilling Zone'. Palette: Sophisticated dark tones with gold accents for premium drinks."

### Prompt 067 : Journal des Sorties de Stock (Protocol Logs)
"Audit log of every bottle served. 'Grace served 2 Champagnes to Table 1 at 21:15'. Filter by server or by table. Minimalist table layout, high-end typography (SF Pro Text). Transparent, honest data view."

### Prompt 068 : Alerte Ravitaillement "Urgent" (Organizer)
"Push notification style alert inside the app. '🚨 Critical: Table 4 (VIP) has empty glasses'. Action: 'Dispatch Protocol'. Glassmorphism overlay with a soft red glow. Clean and effective."

## 🕵️‍♀️ 04. L'Interface "Ange du Protocole" (Field Staff)

### Prompt 069 : Vue Mission du Protocole
"Main task list for the protocol staff. 'Table 1: Serve 2nd round of Loop', 'Table 8: Refill soft drinks'. Simple checkmark buttons to confirm delivery. Focus on speed and discretion. Dark mode for evening events."

### Prompt 070 : Identification du Client VIP (Events)
"Client info screen for protocol. 'Table 1: Uncle of the Groom'. Order history for this specific guest. 'Preference: Whisky with 1 ice cube'. Apple Contacts style, very elegant and informative."

### Prompt 071 : Commande "À la Plume" (Manual Entry for Protocol)
"Quick entry screen to note a special request from a guest. Selection grid of beverages. One-tap validation. No payment mention. Text: 'Compliments of the Host'. Soft blue validation button."

## 🔄 05. Gestion des Boucles (Events Loop)

### Prompt 072 : Configuration de la Boucle VIP
"Special setup for VIP tables. 'Serve 1 bottle of Spirit every 90 mins'. Control switch to 'Auto-approve' or 'Manual-dispatch'. Visual: A gold circular arrow icon. High-end Apple-style settings menu."

### Prompt 073 : Vue d'ensemble des Boucles Actives
"A 'Clock View' showing all ongoing loops in the hall. Timers countdown until next delivery for each table. 'Table 3: 4 mins'. Color-coded urgency. Beautiful, data-dense but readable Apple design."

## 🧊 06. Sécurité & Rapports de Clôture

### Prompt 074 : Rapport de Fin d'Évènement (Summary)
"Final stats for the client (e.g., the Groom). 'Total Bottles Consumed: 145'. 'Total Guests Served: 280'. 'Stock Remaining: 15 bottles'. Clean PDF-style report view inside the app. Top right: 'Share Report'."

### Prompt 075 : Audit de Réconciliation (Stock vs Service)
"Admin reconcile screen. 'Theoretical Stock: 160'. 'Actual Serviced: 158'. 'Difference: 2 bottles (Missing scan?)'. Tool for the organizer to check staff honesty. Clean, factual, professional Apple audit interface."
