# BarPilote : Design Prompts - Batch Serveur (Vigie) 🍏🕵️‍♀️

**Style Global** : Apple-style "Vigie Edition". Focus sur la lisibilité haute-performance (Dark Mode par défaut recommandé pour les environnements de nuit/bar). Contrastes élevés, typographie San Francisco Bold pour les numéros de table, micro-animations pour les nouvelles commandes.
