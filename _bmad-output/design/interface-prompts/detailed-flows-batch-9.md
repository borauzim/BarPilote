# BarPilote : Design Prompts - Batch Flux UX Détaillés 🍏🍊🏃‍♂️💨

**Style Global** : Apple "Flow" Design. Fluidité, feedback immédiat, palette **Orange Vibrant**. Chaque prompt décrit une séquence ou une interaction profonde. Typographie SF Pro, transitions floues, boutons magnétiques.
