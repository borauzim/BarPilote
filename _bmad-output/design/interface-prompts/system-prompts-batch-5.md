# BarPilote : Design Prompts - Batch Système & Edge Cases 🍏⚙️🛠️

**Style Global** : Apple "System" Design. Cohérence, clarté et rassurance. Utilisation de flous de verre (Glassmorphism), de typographies adaptatives et de feedbacks visuels doux pour transformer les erreurs en moments de guidage.

---

## ⏳ 01. États de Chargement (Loading States)

### Prompt 081 : Skeleton Screen - Dashboard Admin
"Sleek Apple-style skeleton loader for the Admin Dashboard. Gray translucent pulses representing cards, graphs, and list items. Zero-lag perception. Pure white background. Soft breathing animation."

### Prompt 082 : Spinner Minimaliste "Apple Watch Style"
"A high-end, smooth loading spinner inspired by watchOS. A ring of dots fading in and out. Below: 'Synchronizing Crate inventory...'. Centered on a dark gray blurred background."

### Prompt 083 : Chargement Progressif d'Image (Blur-up)
"Interface showing a drink menu item loading. Start with a tiny blurred version of the beverage photo, smoothly transitioning to high-res. Soft fade-in effect. Apple Photos app style."

### Prompt 084 : Écran de Chargement Initial "BarPilote"
"App launch screen. Centered minimalist logo. A thin, slow-moving progress line at the very bottom of the screen. Blue accent on white. Elegant, professional, trustworthy."
