# BarPilote : Bibliothèque de Boissons 3D 🍏🥤✨

Ce dossier est destiné à recevoir les visuels 3D haute qualité que vous allez fournir pour l'application BarPilote.

## 📂 Structure du dossier :
- `/beers/` : Pour les bières (Heineken, Primus, Skol, etc.)
- `/spirits/` : Pour les alcools forts (Whisky, Gin, Rhum)
- `/soft-drinks/` : Pour les boissons non-alcoolisées (Fanta, Coca, Eau)
- `/wines/` : Pour les vins et champagnes
- `/cocktails/` : Pour les mélanges spécifiques
- `/logos/` : Pour les logos des marques ou du bar

## 💡 Conseils pour les visuels :
- **Fond** : Transparent (PNG) ou Fond Uni (Blanc/Noir) pour faciliter l'intégration Apple-style.
- **Style** : Rendu photoréaliste avec de la condensation pour un effet "bien frappé".
- **Nommage** : Utilisez des noms simples (ex: `heineken_33cl.png`, `primus_65cl.png`).

Je suis prêt à les intégrer dès que vous les déposez ici ! 🚀
